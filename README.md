# YouTube to YouTube Live Streamer

This system allows users to stream an existing YouTube video as a LIVE broadcast.

### Features:
- Direct RTMP Streaming using FFmpeg.
- **5-Minute Free Trial** logic included.
- Simple REST API.

### Setup:
1. Install FFmpeg on your server.
2. Run `npm install`.
3. Start the server: `node server.js`.
